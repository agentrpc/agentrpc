## TypeScript SDK

The TypeScript SDK provides a simple way to register functions as tools for AI agents.

### Installation

```bash
npm install agentrpc
```

### Basic Usage

```typescript
import { AgentRPC } from "agentrpc";
import { z } from "zod";

const client = new AgentRPC({
  apiSecret: "YOUR_API_SECRET",
});

client.register({
  name: "hello",
  schema: z.object({ name: z.string() }),
  handler: async ({ name }) => `Hello ${name}`,
  // Optional configuration
  config: {
    retryCountOnStall: 3,
    timeoutSeconds: 30,
  },
});

await client.listen();
```

### OpenAI Integration

```typescript
// Get tools formatted for OpenAI
const tools = await client.OpenAI.getTools();

// Pass these tools to OpenAI
const chatCompletion = await openai.chat.completions.create({
  model: "gpt-4o",
  messages: messages,
  tools: tools,
  tool_choice: "auto",
});

// Execute tool calls from OpenAI's response
if (responseMessage.tool_calls && responseMessage.tool_calls.length > 0) {
  for (const toolCall of responseMessage.tool_calls) {
    const result = await client.OpenAI.executeTool(toolCall);
    // Use the result...
  }
}
```

### MCP Server

The TypeScript SDK includes an MCP (Model Context Protocol) server that can be started using:

```bash
npx agentrpc mcp <YOUR_API_SECRET>
```

For more details, visit the [TypeScript SDK GitHub repository](https://github.com/agentrpc/agentrpc/tree/main/sdk-node).

## Go SDK

The Go SDK provides a clean interface for registering functions as tools for AI agents.

### Installation

```bash
go get github.com/agentrpc/agentrpc/sdk-go
```

### Basic Usage

```go
package main

import (
  "context"
  "fmt"

  "github.com/agentrpc/agentrpc/sdk-go"
)

func main() {
  client, err := agentrpc.New(agentrpc.Options{
    APISecret: "YOUR_API_SECRET",
  })
  if err != nil {
    panic(err)
  }

  err = client.Register(agentrpc.Tool{
    Handler: func(input map[string]interface{}) (interface{}, error) {
      name, ok := input["name"].(string)
      if !ok {
        return nil, fmt.Errorf("name must be a string")
      }
      return fmt.Sprintf("Hello %s", name), nil
    },
    Name:        "SayHello",
    Description: "A simple greeting function",
  })
  if err != nil {
    panic(err)
  }

  err = client.Listen()
  if err != nil {
    panic(err)
  }

  // To stop listening
  // client.Unlisten()
}
```

For more details, visit the [Go SDK GitHub repository](https://github.com/agentrpc/agentrpc/tree/main/sdk-go).

## Python SDK

> ⚠️ The AgentRPC Python SDK does **not** currently support registering tools.


```bash
pip install agentrpc
```

### Basic Usage

```python
from agentrpc import AgentRPC

client = AgentRPC(api_secret="YOUR_API_SECRET")

@client.register
def get_weather(location: str) -> dict:
    """Get the weather for a specific location."""
    # Implementation details
    return {"temperature": 72, "condition": "sunny", "location": location}

client.listen()
```

### OpenAI Tools

AgentRPC provides integration with OpenAI's function calling capabilities, allowing you to expose your registered RPC functions as tools for OpenAI models to use.

#### Agents SDK

##### `client.openai.agents.get_tools()`

The `get_tools()` method returns your registered AgentRPC functions as OpenAI Agent tools.

```python
# First register your functions with AgentRPC (Locally or on another machine)

# Attach the tools to the Agent
agent = Agent(name="AgentRPC Agent", tools=agentrpc.openai.agents.get_tools())

result = await Runner.run(
    agent,
    input="What is the weather in Melbourne?",
)

print(result.final_output)

```

#### Completions SDK
##### `client.openai.completions.get_tools()`

The `get_tools()` method returns your registered AgentRPC functions formatted as OpenAI tools, ready to be passed to OpenAI's API.

```python
# First register your functions with AgentRPC (Locally or on another machine)

# Then get the tools formatted for OpenAI
tools = agentrpc.openai.get_tools()

# Pass these tools to OpenAI
chat_completion = openai.chat.completions.create(
  model="gpt-4-1106-preview",
  messages=messages,
  tools=tools,
  tool_choice="auto"
)
```

##### `client.openai.completions.execute_tool(tool_call)`

The `execute_tool()` method executes an OpenAI tool call against your registered AgentRPC functions.

```python
# Process tool calls from OpenAI's response
if chat_completion.choices[0].tool_calls:
  for tool_call in response_message.tool_calls:
    client.openai.execute_tool(tool_call)
```
